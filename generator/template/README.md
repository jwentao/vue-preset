# vue-cli3-template

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### 发布
```
npm run build-formal  # 打包正式环境代码
npm run build-test    # 打包测试环境代码
```

See [Configuration Reference](https://cli.vuejs.org/config/).
